# Public Use Cases

These examples are public-facing scenario pages for GitHub visitors.

They are meant to answer two questions quickly:
- what kind of image does `attention` work well on?
- what kind of hook and draft does it try to produce?

Current examples:
- [fashion / outfit focus](./fashion-lookbook.md)
- [accessories / detail focus](./accessories-detail.md)
- [cafe / mood shot focus](./cafe-detail.md)

If you want to submit another scenario:
- use the [use case issue form](https://github.com/rickqdev/attention/issues/new/choose)
- or open a GitHub Discussion with the image pattern you care about
