# attention Roadmap

`attention` is being shaped around one core promise:

Turn an image into the hook, question, and first draft a creator can actually build on.

## Current Focus

- make the browser demo easier to understand in under 60 seconds
- make the repo easier for creators to try and for developers to integrate
- expand public examples that show different image patterns and content angles

## Next Milestones

## 1. Better First-Time Creator Flow

- clearer upload -> hook -> draft narrative
- stronger example coverage for fashion, accessories, and cafe content
- faster guidance for what context improves the draft

## 2. Cleaner Developer Integrations

- tighter API and MCP quickstarts
- better example requests and response walkthroughs
- clearer extension and workflow integration guidance

## 3. Feedback Loop On GitHub

- GitHub Discussions for Q&A, use cases, and announcements
- structured issue intake for bugs, feature requests, and new scenarios
- release notes written around user-visible value, not only internal implementation

## Out Of Scope For v1

- automatic publishing
- account operations
- comment monitoring
- guaranteed virality claims
