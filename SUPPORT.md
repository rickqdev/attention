# Support

## Where To Ask

- Questions and usage help: [GitHub Discussions](https://github.com/rickqdev/attention/discussions)
- Bugs: [open a bug report](https://github.com/rickqdev/attention/issues/new/choose)
- Feature ideas and new scenarios: [open an issue or use case submission](https://github.com/rickqdev/attention/issues/new/choose)

## Before You Open A Ticket

- confirm which interface you used: demo, CLI, API, MCP, or skill
- include the provider name you selected
- include the exact error message if one exists
- remove real keys, private images, and sensitive output before posting

## Safe Sharing

Please do not post:
- API keys
- private customer or creator data
- images you do not have permission to share

If your issue depends on private material, describe the pattern instead of attaching the asset.
